# Архитектура
:recycle: Используется стандартная ООП архитектура с менеджерами

:recycle: Точка входа GameEntryPoint - ручная инициализация всех зависимостей.

:recycle: DroneStateMachine - AI дронов с централизованным DroneStateContext.

:recycle: EventBus - глобальная коммуникация происходит через него.

:recycle: В основном менеджеры делегируют отвественности, создают объекты и прокидывают зависимости.

:recycle: Для UI это вьюшки которые вызывают EventBus или наоборот реагируют на него.
